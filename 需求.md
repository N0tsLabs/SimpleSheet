# SimpleSheet - 轻量级智能表格框架

## 项目概述

| 项目名称 | SimpleSheet |
| --- | --- |
| 技术栈 | TypeScript（纯原生，零依赖） |
| 目标 | 轻量、易用、可扩展的 Excel 风格表格框架 |
| 参考产品 | 腾讯文档智能表格、飞书多维表格 |
| 发布平台 | npm |
| 兼容性 | TypeScript / JavaScript 均可引用 |
| 运行环境 | 现代浏览器（Chrome、Firefox、Safari、Edge） |

---

## 痛点分析

市面上存在的表格框架存在以下问题：

| 问题类型 | 具体表现 |
| --- | --- |
| 过于庞大 | 如 Handsontable、ag-Grid，包体积大、学习成本高 |
| 稳定性差 | 一些开源项目迭代频繁，bug 多，API 不稳定 |
| 功能受限 | 简单表格库无法满足复杂业务场景 |
| 扩展性差 | 自定义渲染困难，无法轻松渲染按钮、图片等组件 |
| 集成麻烦 | 数据导入导出不够灵活，与后台数据对接繁琐 |

### 典型场景需求

**场景一：后台管理系统**
- 把列表页当做 Excel 使用
- 自由定义字段和字段类型（文本、图片、链接、文件等）
- Excel 风格的批量编辑、删除
- 记住表格配置（列宽、排序、筛选条件等）

**场景二：自定义单元格渲染**
- 在单元格内渲染 HTML 内容
- 渲染按钮、开关、进度条等交互组件
- 支持自定义编辑器

## API 设计

### 初始化
```typescript
import { SimpleSheet } from 'simple-sheet';

const sheet = new SimpleSheet('#container', {
  // 列定义
  columns: [
    { 
      key: 'name', 
      title: '名称', 
      width: 150,
      type: 'text',
      readonly: false,
      renderer: CustomRenderer,
      editor: CustomEditor,
    },
    { key: 'price', title: '价格', width: 100, type: 'number' },
    { key: 'date', title: '日期', width: 120, type: 'date' },
  ],
  
  // 数据
  data: [],
  
  // 配置
  options: {
    rowHeight: 32,
    headerHeight: 40,
    readonly: false,
    allowMultiSelect: true,
    showRowNumber: true,
    showCheckbox: false,
    theme: 'light',
  }
});
```

### 主要方法
```typescript
// 数据操作
sheet.loadData(data, options?)
sheet.getData(): any[]
sheet.clearData()

// 单元格操作
sheet.getCellValue(row, col): any
sheet.setCellValue(row, col, value)
sheet.getCellMeta(row, col): CellMeta
sheet.setCellMeta(row, col, meta: Partial<CellMeta>)

// 选择操作
sheet.getSelection(): Selection
sheet.setSelection(startRow, startCol, endRow?, endCol?)
sheet.clearSelection()
sheet.getSelectedData(): any[]

// 行列操作
sheet.insertRow(index, data?, count?)
sheet.deleteRow(index, count?)
sheet.insertColumn(index, column, count?)
sheet.deleteColumn(index, count?)
sheet.setColumnWidth(index, width)
sheet.setRowHeight(index, height)

// 滚动
sheet.scrollToCell(row, col)
sheet.scrollToRow(row)
sheet.scrollToColumn(col)

// 历史
sheet.undo()
sheet.redo()
sheet.clearHistory()

// 导入导出
sheet.importCSV(csvString, options?)
sheet.exportCSV(options?): string
sheet.importJSON(json, mapping?)
sheet.exportJSON(): any[]

// 销毁
sheet.destroy()
```
